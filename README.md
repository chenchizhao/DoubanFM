Douban FM
========
Douban FM客户端Linux版

目前只有最基本的播放功能

Qt version: 4.6.3

依赖：phonon libphonon-dev libqt4-phonon

## 安装方法
```bash
qmake doubanfm.pro
make
```

直接运行doubanfm即可

在装有多块声卡的机器上如果听不到声音，可以试着切换声卡驱动程序。在System Settings里，找到Audio Output-Music，调高Internal Audio Analog Stereo的优先级。
